## 📖 О репозитории

Этот репозиторий создан для систематического изучения Spring Framework.

### Цели репозитория

- ✅ Изучить IoC и DI на практике
- ✅ Освоить различные способы конфигурации Spring
- ✅ Понять механизмы внедрения зависимостей
- ✅ Научиться работать со свойствами и внешними ресурсами
- ✅ Создать базу для изучения Spring Boot, Spring Data, Spring Security
- ✅ Вести документацию изученных концепций
